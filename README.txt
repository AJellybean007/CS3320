To run configuration from terminal:
- In a cmd, have the working directory be ./CS3320-main/client and enter
-- npm start
- In a cmd, have the working directory be ./CS3320-main/server and enter
-- npm run dev

To access database of your choosing 
- Go to line 6 of db.js to enter a mongodb access uri
-- db.js can be found at ./CS3320-main/server/config/db.js
